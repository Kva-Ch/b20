Some changes have been made to Bugzilla recently

ID Change Summary Change Description When
1020709 Governance Approved flags for both EHF and SCI 'Governance_Approved' flag is replaced with two new flags, i.e. 'Governance_Approved_EHF' and 'Governance_Approved_SCI'. When one among them is set to '?', the other flag is also set to '?' 5-24-2021
1015489 Adding "Quick file Modular BIG-IP bug" and "Quick file VELOS/F5OS bug" to home page On the homepage users can now quick file Modular BIG-IP bug and VELOS/F5OS bug 5-9-2021
993977 PM/PO Owner field is now available in bulk edit fields Users can now change the PM/PO owner field of several bugs at once, by using the feature 'Change Several Bugs at Once'. 2-15-2021
993973 Component page now lists PM/PO Owner There is a lot of confusion now a days on who is PM/PO owner for different components. Currently this data is not visible to everybody. Now this field visible to everyone, so that it will be easy for bugzilla users to trace the PM/PO owner for specific component. 2-15-2021
975513 Jira Issue Id is now editable via UI. Previously bugspray tool was used to update JIRA Issue Id field in bugzilla for bugs which are tracked in JIRA, now the JIRA Issue Id is made readable and writable so that one can easily update it using the UI itself. 12-20-2020
858397 Add who the bug was assigned to into history when it's created For reporting purposes, when a new bug is filed, the assignee information (Assigned To, QA Contact, PO/PM Owner) are added to the history. 12-7-2019
858393 Assigned date field not always set Fixed some logic errors to make sure that it always gets set when the assignee changes, this includes when the bug is first opened (at which point it will be the same as the bugs creation time since it's assigned to someone then) 12-7-2019
858389 Update PM Owner field label PM Owner has been renamed to PO/PM Owner, note that all existing search queries will still work, this is simply updating the label, not the underlying field/data. 12-7-2019
726354 PM Owner is now searchable and shows up by email address in search results Now when performing a search you can go to "Search By People" and search for the PM Owner. Additionally if you include the PM Owner column in your results it will show the users email address instead of their user id number like it previously did. 6-30-2018
706216 New PM Owner field for improvements and requirements There is now a PM Owner field which will be used for requirements and improvements that are tracked in bugzilla. This is a required field for improvements or requirements. Also fixed the alignment of the quick file buttons on the home page. 2-10-2018
